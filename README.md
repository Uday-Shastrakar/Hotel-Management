# Hotel-Management
# Hotel Booking Backend

Spring Boot backend for Hotel Booking System.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- REST APIs
- Lombok
- Validation

## Features
- User Management
- Hotel Search by City
- Room Management
- Booking System
- Prevent Double Booking
- Pagination & Filtering
- Exception Handling
- Clean Architecture

## Modules
- User
- Hotel
- Room
- Booking

## Architecture
Controller → Service → Repository

## APIs
- Create User
- Add Hotel
- Add Room
- Search Hotels
- Book Room
- Cancel Booking
- Get Bookings

## Future Improvements
